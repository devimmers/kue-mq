## Getting Started

Install the module with: `npm install kue-mq`

```js
var kue-mq = require('kue-mq');
kue-mq.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g kue-mq
$ kue-mq --help
$ kue-mq --version
```
